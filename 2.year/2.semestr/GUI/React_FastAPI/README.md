## 📜 Popis projektu

Tento projekt slouží jako vyučovací materiál pro seznámení s React a FastAPI.
Projekt je prací jiné dvojice studentů v rámci kurzu "Programování pro GUI".

## ⚙️ Funkce

- Seznámení s React
- Hlubší práce s FastAPI
