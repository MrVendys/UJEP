## 📜 Popis projektu

Tento projekt slouží jako vyučovací materiál pro seznámení s SQLAlchemy. 
Pojekt je prací jiné dvojice studentů v rámci kurzu "Programování pro GUI"

## ⚙️ Funkce

- Seznámení s SQLAlchemy
- Práce s SQLite
