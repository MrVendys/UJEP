# How to aktivovat tuto práci.
+ udělat si venv `python -m venv venv`
+ nainstalovat knihovny z requirements.txt `pip install -r requirements.txt`
+ `docker-compose up`

# Co tam funguje
+ login (testovací uživatelé jsou `bao` nebo `venca`)
  + jestliže hodí error typu unauthorized něco něco, tak zadat odkaz na `locahost:5000/logout`
+ přidávání spellů či itemů
